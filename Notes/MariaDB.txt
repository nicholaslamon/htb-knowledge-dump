# Notes on MariaDB Commands/Useful Information

## Commands:

* `mysql --host=[host address] -u root -p` - Connect to a databse anonymously.
* `show databases` - List all MariaDB databases.
* `use [database]` - Connect to a specific database.
* `show tables` - Lists all tables within a database.
* `select database()` -Lists what database you are currently in.
* `select [item]` - Displays the data of an item within a table.

## Useful Information

* Always end commands with `;`, otherwise MariaDB gets mad at you.
